## 项目简介

- 该项目是一个**纯PDF查看器**
  - 写该项目纯属是为了让我的小设备能打开PDF文件
- 项目使用 **Android Studio** 打造
  - 以 **Kotlin** 和 **XML** 制作 ~~其实代码基本是deepseek写的~~

## 开源原因

- 学习 Git 和 GitHub
- 为一些不想写代码但想使用这种软件的人提供 APK [下载APK](https://www.baidu.com)
